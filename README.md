# Monad NFT Mint UI

Simple frontend interface to mint an ERC-721 NFT deployed on Monad Testnet.

## Features
- Connect wallet (MetaMask)
- Mint NFT from smart contract
- Display transaction hash
- Simple and clean UI

## Tech Stack
- React
- ethers.js
- Monad Testnet
