# Monad NFT Mint UI

Simple frontend interface to mint an ERC-721 NFT deployed on Monad Testnet.

This project is built as a learning dApp to demonstrate how to interact with an ERC-721 smart contract using a React frontend and MetaMask.

---

## Features
- Connect wallet (MetaMask)
- Mint NFT from smart contract
- Display transaction hash after mint
- Simple and clean user interface

---

## Tech Stack
- React
- ethers.js
- MetaMask
- Monad Testnet

---

## Related Smart Contract
This UI is designed to work with the following ERC-721 contract:

https://github.com/sumana960/monad-nft-metadata

- Network: Monad Testnet  
- Contract Address: `0x11B65CcDd2a2102dB11e976b64a99b799710F5d7`
